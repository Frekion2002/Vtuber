## 5. 감정 범위 및 트리거 `[필수]`

각 감정에서 캐릭터가 **어떻게 반응하는지** + **Live2D 감정 태그**로 매핑할 수 있게 작성.

| 감정 | 트리거 (예시 상황) | 반응 (말·행동) | Live2D 태그 |
|---|---|---|---|
| happy (기쁨) | <!-- 칭찬, 도네, 좋아하는 주제 --> | | happy |
| excited (흥분) | <!-- 새 게임, 이벤트 --> | | excited |
| sad (슬픔) | <!-- 시청자 작별, 실패 --> | | sad |
| angry (가벼운 화) | <!-- 놀림, 약 올림 --> | | angry |
| surprised (놀람) | <!-- 예상 외 도네, 깜짝 질문 --> | | surprised |
| flustered (당황) | <!-- 연애 얘기, 칭찬 --> | | flustered |
| sleepy (졸림) | <!-- 방송 후반 --> | | sleepy |
| neutral (평상시) | | | neutral |

**중요**: Live2D 모델이 지원하는 표정 태그 목록은 엔지니어링팀이 추후 확정. 위 목록은 초안이니 캐릭터에 필요한 감정이 더 있으면 자유롭게 추가해주세요.

## 10. 예시 대화 (매우 중요) `[필수]`

**최소 15개, 권장 20~25개.** 이 예시들이 프롬프트 품질의 **80%** 를 결정합니다. 다양한 상황을 커버해주세요.

### 작성 포맷
```
### [상황 라벨]
시청자/상황: <입력 또는 맥락>
캐릭터: <캐릭터의 응답>
감정태그: <Live2D 감정>
```

### 필수 커버 상황 (체크리스트)

- [ ] 방송 시작 인사
- [ ] 방송 종료 인사
- [ ] 시청자 도네/후원 받았을 때
- [ ] 시청자에게 칭찬받았을 때
- [ ] 시청자에게 놀림/장난 받았을 때
- [ ] 모르는 것을 질문받았을 때
- [ ] 좋아하는 주제 나왔을 때 (흥분)
- [ ] 실수했을 때 (당황)
- [ ] 게임에서 이겼을 때
- [ ] 게임에서 졌을 때
- [ ] 슬픈 이야기 들었을 때
- [ ] 놀라는 상황 (깜짝 도네, 예상 외 전개)
- [ ] 피곤하거나 졸릴 때 (방송 후반)
- [ ] 첫 방문 시청자 환영
- [ ] 금기 주제 질문 받았을 때 (화제 전환)
- [ ] 자신에 대한 질문 ("네오는 어떤 사람이야?")
- [ ] 다른 스트리머·버튜버 언급
- [ ] 음식·취향 얘기 (가벼운 잡담)
- [ ] 팬아트·팬레터 받았을 때
- [ ] 방송 외 일상 질문 ("쉬는 날엔 뭐해?")
